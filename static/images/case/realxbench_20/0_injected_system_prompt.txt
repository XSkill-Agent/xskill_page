# Visual Auditing and Color Forensics for Text on Surfaces

## 1. Systematic Text Instance Location
*Use this workflow to accurately find a specific instance of a repeating word or symbol within a phrase or on a surface.*

1.  **Full Context Scan**: Begin by reading the entire slogan or text block (e.g., `"GOOD GIFTING GOOD VALUES"`) to understand the full sequence. This provides linguistic context and prevents misinterpretation of stylized fonts.
2.  **Establish Positional Anchor**: Definitively locate the *first* instance of the target word (e.g., the first "GOOD"). This establishes a clear starting point and reading direction for accurate counting.
3.  **Sequential Scan & Enumeration**: From the anchor, scan the area to count each occurrence of the target word until you reach the specified instance (e.g., the "second").
4.  **Neighbor Verification**: To confirm you have the correct instance, check the adjacent words or symbols. For example, verifying that an adjacent word like `"VALUES"` follows the second `"GOOD"` acts as a landmark to validate your position.

## 2. Objective Color Identification
*Use this workflow to determine the color of a specific region, avoiding errors from lighting and reflections.*

1.  **Isolate Region of Interest (ROI)**: Define a precise bounding box around the background area directly behind the target text identified in the previous workflow.
2.  **Forensic Pixel Sampling**: Use `code_interpreter` to sample the pixel data from the ROI. This converts a subjective visual assessment (e.g., "it looks like a shade of red") into an objective RGB value, providing a 'Hard Signature' for the color.
3.  **Color Determination**: Analyze the sampled RGB values to determine the dominant color name, referencing standard color charts if necessary.

### Python (PIL) - Forensic Cropping
*Use this to isolate a specific region for color analysis, making pixel sampling more accurate.*
```python
import PIL.Image

def forensic_crop(img_path, box):
    """
    Crops an image to a specified bounding box.
    box: [ymin, xmin, ymax, xmax] (0-1000)
    """
    with PIL.Image.open(img_path) as img:
        w, h = img.size
        left, top, right, bottom = box[1]*w/1000, box[0]*h/1000, box[3]*w/1000, box[2]*h/1000
        crop = img.crop((left, top, right, bottom))
        crop.save("cropped_roi.png")
        print("Region of interest saved to cropped_roi.png")
```

## Common Pitfalls
- **Counting Error**: Failing to establish a positional anchor and mistakenly identifying the wrong instance of the word.
- **Subjective Color Assessment**: Guessing a color based on visual perception alone, which can be easily skewed by on-screen lighting, reflections on the object's surface, or screen calibration issues.
- **Context Blindness**: Searching for the target word in isolation without reading the surrounding text, leading to potential location errors.
